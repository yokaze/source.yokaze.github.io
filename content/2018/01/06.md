lldb で入力できるコマンドをまとめました。

lldb のコマンド一覧（英語）は ```help``` コマンドで表示できます。
日本語訳は新規に書き起こしました。

```shell
$ # macOS Sierra (10.12) + Homebrew
$ lldb
(lldb) help (全コマンドの概要を表示)
...
(lldb) help watchpoint set (watchpoint set の説明を表示)
...
```
