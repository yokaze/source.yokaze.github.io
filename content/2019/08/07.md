TensorFlow 2.0 の `Variable` を `ndarray` に変換するには `tensorflow.Variable.numpy()` を使います。

```python
import tensorflow

x = tensorflow.Variable(tensorflow.random.uniform([4, 4]))
print(x)

y = x.numpy()
print(y)
```
