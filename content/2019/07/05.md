lldb で関数の呼び出し履歴（バックトレース）を表示するには、`bt` コマンドを使います。

```cpp
#include <stdio.h>

void sample1()
{
    printf("sample 1\n");
}

void sample2()
{
    sample1();
}

void sample3()
{
    sample2();
}

int main(void)
{
    sample3();
    return 0;
}
```

$ lldb sample
